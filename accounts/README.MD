# 🏠 Home Service Booking System

A modern Home Service Booking System developed using **Python, Django, HTML, CSS, JavaScript, Bootstrap, and SQLite**. The application connects customers with skilled workers such as electricians, plumbers, AC repair technicians, mobile repair technicians, carpenters, and cleaning service providers.

---

# 📌 Features

## 👤 Customer Module

- Customer Registration & Login
- View Available Workers
- Book Home Services
- Live Worker Tracking
- View My Bookings
- Cancel Booking
- OTP Verification
- Online/Cash Payment
- Review & Rating
- Customer Profile Management
- Customer Support

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/PABITRASWAIN619/homeservicewebsite.git
```

Go to project folder

```bash
cd Home-Service-Booking-System
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Environment

Windows

```bash
venv\Scripts\activate
```

Linux / Mac

```bash
source venv/bin/activate
```

Install Requirements

```bash
pip install -r requirements.txt


Run Migrations

bash
python manage.py makemigrations
python manage.py migrate


Run Server bash
python manage.py runserver


Open Browser


http://127.0.0.1:8000/
